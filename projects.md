# My Projects

Here are some of the engineering projects I've worked on. Each project represents a learning journey and a step toward solving real-world challenges.

## IoT Environmental Monitoring Network

**Technologies:** ESP32, DHT22 sensors, WiFi, Raspberry Pi, Python, InfluxDB
**Description:** A distributed sensor network that monitors temperature and humidity across multiple locations. The system features real-time data collection, trend analysis, and automated alerting when environmental conditions exceed defined thresholds.
**Key Features:**
- Low-power wireless sensor nodes
- Central data aggregation and storage
- Web-based monitoring dashboard
- Expandable architecture for additional sensors

## Smart Home Automation Controller

**Technologies:** Arduino, Relay modules, Mobile app development, REST API
**Description:** An integrated home automation system that allows remote control of lights, appliances, and environmental systems. The project includes both hardware design and software development for seamless user interaction.
**Key Features:**
- Voice control integration
- Scheduled automation routines
- Energy usage monitoring
- Mobile app interface

## Wireless Sensor Data Logger

**Technologies:** LoRa communication, Solar power, Weatherproof enclosures, Data visualization
**Description:** A battery-powered, solar-recharged sensor platform designed for remote environmental monitoring. The system can operate independently for months while transmitting data over long distances.
**Key Features:**
- Ultra-low power consumption
- Long-range wireless communication
- Weather-resistant design
- Automated data logging and transmission
